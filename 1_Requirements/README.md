# Requirements

## 1 Software requirements

a) Code Blocks IDE/Visual Studio editor.\
b) MingW compiler.\
c) Microsoft word.\
d) Notepad.

## 2 Hardware requirements

a) Operating System:  Windows or Linux or Mac\
b) Processor       :  Intel Core 32 bit or above\
c) RAM             :  4GB or above
